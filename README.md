# Kushal Khandelwal - Personal Portfolio

A minimal, clean personal portfolio website with blog functionality.

## Features
- Clean, responsive design
- About section
- Blog system with Markdown support
- Static site generation

## Setup
1. Install dependencies: `npm install`
2. Add blog posts as `.md` files in `/blogs` folder
3. Build the site: `npm run build`
4. Preview locally: `npm run serve`

## Deployment
The generated static site in `/dist` can be deployed to any static hosting service.
